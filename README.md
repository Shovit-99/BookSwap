# BookSwap 📚🔁

BookSwap is a full-stack web application designed for listing, browsing, and exchanging books. It features an interactive user dashboard, profile management, and a centralized book marketplace.

## 🚀 Live Demo
* **Frontend:** [https://book-swap-nfhu.vercel.app](https://book-swap-nfhu.vercel.app)
* **Backend API:** [https://bookswap-backend-7daz.onrender.com](https://bookswap-backend-7daz.onrender.com)

---

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Deployment:** Vercel (Client), Render (Server), MongoDB Atlas (Database)
